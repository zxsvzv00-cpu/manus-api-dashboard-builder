# Manus API Dashboard Builder

A Manus Skill for building beautiful, production-ready API dashboards with Next.js, Tailwind and shadcn/ui.

This skill gives Manus everything it needs to scaffold, design, and ship a complete dashboard for any REST API.

### ✨ Features
- **Auto API Client**: Generate typed fetch client from OpenAPI spec
- **Design System Ready**: Built-in design tokens, dark mode, responsive layout
- **Pages Included**: Auth, Overview, API Keys, Usage, Logs, Settings
- **Production Grade**: Loading states, error handling, pagination, search

### 📦 Installation

```bash
manus skill install zxsvzv00-cpu/manus-api-dashboard-builder
